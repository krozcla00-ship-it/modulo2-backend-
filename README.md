# 📦 ALY-catalogo de ropa deportiva para mujeres

Catalogo paraventa de ropa deprotiva exclusiva para mujeres

## 👥 Integrantes del Grupo
* edgar jose alapez paez

## 🚀 Tecnologías Utilizadas
* HTML5 (Estructura semántica)
* CSS3
* Bootstrap 5 (Diseño responsivo)
* JavaScript (Lógica de filtrado y validación)

## ⚙️ Funcionalidades Principales
* **Filtrado dinámico:** Permite buscar recursos por categorías mediante JavaScript.
* **Vista detallada:** Sección interactiva para ver la información completa de cada recurso.
* **Formulario de contacto:** Validación de campos en tiempo real antes del envío.
* **Diseño Responsivo:** Adaptado para celulares, tablets y computadoras.
